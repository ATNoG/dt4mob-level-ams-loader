## Installation

**1. Clone the repository**

```bash
git clone https://github.com/ATNoG/dt4mob-level-ams-loader.git
cd dt4mob-level-ams-loader
```

**2. Install dependencies**

```bash
uv sync
```

This installs all required Python packages in an isolated environment.

**3. Create configuration file**

```bash
cp config.example.toml config.toml
```

**4. Configure the service**

Edit `config.toml` with your environment-specific settings. See the [Configuration](#configuration) section for all available options.
